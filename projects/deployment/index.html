Data science is fine - but how do you actually make it usable for the business user?

This small project is to give an example of a modern data science product using methods such as modularization, data gathering from external sources, continuous re-training of a machine learning model, publishing and visualizing via an API - all hosted on AWS.

The main idea is to build everything around an API, and use that API for (A) controlling/triggering main fuctions such as unpdating the database, re-training the machine learning model and… (B) …making data and results available through jsons. As a result, any theoretical application accross all devices can access this data science product - as long it is connected to the web!

Exemplary application: A simple dashboard that shows live water levels as well as most recent predictions
